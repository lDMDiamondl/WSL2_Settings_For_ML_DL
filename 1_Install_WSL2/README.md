Windows 기능 켜기/끄기에서 네모 표시를 선택한 뒤 아래의 명령어를 PowerShell에 입력하기

```powershell
# WSL2를 기본 버전으로 변경
wsl --set-default-version 2

# Ubuntu 22.04 설치
wsl --install -d Ubuntu-22.04
```
